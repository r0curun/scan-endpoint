
# Scan-Endpoint

Scans: HTML, JS, CSS, images, fonts, etc.

Extracts paths: /admin, ./api, ../config, /section
Displays real-time logs
Copyable results at the end
Junk filter
Compatible with all modern browsers



## How to use it:

```bash
Copy all the code from the file.

Create a new bookmark in your browser (Works in all current browsers as of 2025).
Paste it as the bookmark URL.
Give it a name like: DirScan or whatever you prefer.
Visit any website → click on the bookmark → done!

```
